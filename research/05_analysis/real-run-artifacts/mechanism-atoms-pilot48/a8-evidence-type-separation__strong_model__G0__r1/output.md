**Claims Classification by Evidence Type**

| Claim | Evidence Type | Rationale |
|-------|---------------|-----------|
| “The patient’s blood pressure was 120/80 mmHg.” | **Directly measured result** | This is a single, objective measurement from a specific observation. |
| “The patient likely has early-stage renal impairment.” | **Inference from multiple slices** | This conclusion is drawn by synthesizing several pieces of data (e.g., lab trends, imaging, symptoms). |
| “There is a potential risk of contrast-induced nephropathy.” | **Unresolved risk** | This identifies a possible adverse outcome that has not yet occurred and requires monitoring. |
| “Schedule a follow-up renal function test in 3 months.” | **Recommended next action** | This is a proposed future step, not an established fact about the patient’s current state. |

**Note:** Recommendations are clearly separated from extracted facts and are not presented as established evidence.
