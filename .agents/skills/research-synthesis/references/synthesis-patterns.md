# Synthesis Patterns

## Available Methods

### 1. Thematic Synthesis
**When**: Multiple sources discuss similar topics; need to find common themes.
**How**: Code evidence by topic → group into themes → identify patterns → assess strength.
**Output**: Theme map with evidence counts and confidence.

### 2. Comparative Matrix
**When**: Comparing multiple objects (tools, papers, competitors, policies).
**How**: Define comparison dimensions → fill matrix from evidence → identify gaps and leaders.
**Output**: Comparison table with evidence references per cell.

### 3. Gap Analysis
**When**: Looking for what's missing in a field, market, or plan.
**How**: Map current state from evidence → define ideal state → identify gaps → prioritize.
**Output**: Gap list with severity and opportunity assessment.

### 4. Contradiction Analysis
**When**: Sources disagree on key claims.
**How**: List contradicting evidence pairs → assess authority of each → determine resolution or accept ambiguity.
**Output**: Contradiction matrix with resolution status.

### 5. Causal Chain
**When**: Need to establish cause-effect relationships.
**How**: Identify proposed causes → trace evidence chains → assess alternative explanations → grade confidence.
**Output**: Causal diagram or narrative with evidence links.

### 6. Decision Matrix
**When**: Comparing options/alternatives for a decision.
**How**: Define criteria → weight criteria → score options against evidence → calculate weighted scores.
**Output**: Scored decision matrix.

### 7. Confidence Grading
**When**: Need to communicate certainty levels in findings.
**How**: Assess evidence volume, source quality, agreement level → assign grade per finding.
**Output**: Findings list with confidence grades (A/B/C/D).

## Method Selection Guide

| Research Type | Default Methods |
|---|---|
| Scientific | Thematic synthesis + Gap analysis + Contradiction analysis |
| Product | Comparative matrix + Thematic synthesis + Decision matrix |
| Planning | Gap analysis + Causal chain + Decision matrix |
