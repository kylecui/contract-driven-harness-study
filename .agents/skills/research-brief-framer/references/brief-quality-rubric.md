# Brief Quality Rubric

## Mandatory Elements (all must be present)

| Element | Criterion |
|---|---|
| Core question | Single, clear, answerable question |
| Sub-questions | At least 3, each actionable |
| Research type | Explicitly classified |
| Scope (In) | Concrete boundaries |
| Scope (Out) | Explicit exclusions |
| Expected output | Named deliverable format |
| Evidence requirements | Minimum standards defined |

## Quality Scoring

### A — Ready for execution
- All mandatory elements present and specific
- Questions are falsifiable or answerable
- Scope is tight enough to complete in stated constraints
- Evidence requirements are measurable

### B — Minor gaps
- All mandatory elements present
- 1-2 elements are vague but recoverable
- Can proceed with stated assumptions

### C — Needs revision
- Missing 1-2 mandatory elements
- Questions too broad or unmeasurable
- Scope allows unbounded work

### D — Incomplete
- Missing 3+ mandatory elements
- No clear question or deliverable
- Cannot proceed without rework

## Common Failures

1. **No question, only topic** — "研究一下AI安全" is not a brief
2. **Scope too broad** — "所有相关文献" is not bounded
3. **No evidence standard** — leads to cherry-picking
4. **No output form** — research drifts without end state
5. **Mixing brief with report** — brief defines the question, not the answer
